HTML List

There are 3 types of list.

1) Ordered or Number list (using OL tag)

2) Unordered or Bullet list (using UL tag)

3) Description list (Using DL tag)

Ordered list

  2. CSS
  3. JAVASCRIPT

Unordered list

  • JAVASCRIPT
